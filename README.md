#  UniMarket

## Descripción
UniMarket es una plataforma web diseñada para que estudiantes universitarios puedan promocionar y vender sus productos dentro de su comunidad. Facilita la conexión entre compradores y emprendedores de manera rápida y sencilla.

## Objetivo
Brindar un espacio exclusivo para estudiantes emprendedores, donde puedan dar a conocer sus productos sin perderse entre publicaciones de redes sociales.

##  Funcionalidades
-  Búsqueda de productos por nombre
-  Filtro por categorías
-  Visualización de productos
-  Perfil de usuario
-  Sistema de notificaciones
-  Mensajes entre usuarios

##  Tecnologías utilizadas
- HTML
- CSS
- JavaScript
