## 📖 Descripción General del Proyecto

**UniMarket** es una red social universitaria diseñada para conectar a los estudiantes de la Universidad de Colima. La plataforma permite una interacción dinámica mediante:

* Publicación de contenido multimedia (imágenes, videos, GIFs).
* Sistema de interacciones (likes, dislikes y comentarios).
* Comunicación en tiempo real a través de chat privado.
* Gestión de perfil y sistema de notificaciones.

## 🗄️ Base de Datos y Tecnología

### Stack Tecnológico
* **Frontend:** HTML5, CSS3, JavaScript.
* **Backend:** PHP (Lógica y API endpoints).
* **Database:** MySQL.
* **Correo:** PHPMailer para recuperación de acceso.

### Estructura SQL (`/config/`)
| Tabla | Propósito | Archivo SQL |
| :--- | :--- | :--- |
| **Usuarios** | Registro y autenticación | `sistema_login.sql` |
| **Publicaciones** | Posts del feed | `publicaciones.sql` |
| **Comentarios** | Feedback de usuarios | `comentarios.sql` |
| **Reacciones** | Control de Likes/Dislikes | `reacciones.sql` |
